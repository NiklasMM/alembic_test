A unit test framework for alembic migrations
===============================

version number: 0.0.1

Overview
--------

A unit test framework for alembic migrations

Installation / Usage
--------------------

To install use pip:

    $ pip install alembic_test


Or clone the repo:

    $ git clone https://github.com/niklasMM/alembic_test.git
    $ python setup.py install
